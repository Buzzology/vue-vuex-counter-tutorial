# vue-3-vuex-counter

## Setup
Install Vuex: `npm install vuex@next`  
Install Axios: `npm install axios`  


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
